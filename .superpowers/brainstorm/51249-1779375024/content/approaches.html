Select a Visual Approach for the Dashboard

We want to balance the "neon-on-cream" aesthetic of the main app with the dark-background requirements of the generated SVGs.

A

Neon-on-Cream Cyberpunk (Recommended)

Blends with the app's primary theme: clean cream surfaces with vibrant hot pink, purple, and cyan glowing accents. The SVG preview containers themselves remain dark/black to preserve contrast, but cards and dashboards use the signature cream background with thin colored borders.

B

Pure Retro-Terminal (Dark Mode Hybrid)

The entire `/robot` sub-page is styled with a dark, high-contrast terminal design (almost like a dashboard from a command center), overriding the cream style. This matches the neon SVGs perfectly and feels like a hacker control panel, but represents a departure from the rest of the application's light-cream design.

C

Minimalist Technical Dashboard

Focuses on raw numbers and tables. Clean grid lines, minimal color, and maximum data density. Gradients are only used for the scores, and SVGs are presented in standard flat boxes. Fits closely with traditional SaaS styling.
